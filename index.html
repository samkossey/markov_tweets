Markovian Tweet Generator

Given a text file, this program will generate a tweet in the writing style of the text.

How it Works

The program stores the probabilities of how likely a word is to appear after a given word based on the given text. The more often a word succeeds a given word in the input file, the more likely it is to be added to a tweet after the given word. In this way, the program honors the word orderings that were used in the input file. The next word in the tweet is dependent only on the current word. Longer text files produces the most realistic tweets based on writing style because the program has more data to work from.

Security Considerations

This project was reviewed for its use of the Memory Management (MEM) and Characters and Strings (STR) rules and recommendations from the SEI CERT C Coding Standard and CERT C++ Coding Standard, but as the first project of the semester, was not initially coded with these specifically in mind. Instead we relied on any security measures we were currently aware of as pre-assessment.
